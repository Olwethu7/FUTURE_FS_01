# Task 2 – Client Lead Management System (CRM)

## Overview
This project is a **Client Lead Management System** built during the FUTURE_FS_01 Internship.  
It allows users to add, manage, and view client leads in a professional web interface.

## Features
- Add new client leads with Name, Email, Source, and Notes.
- View all leads in a sortable and responsive table.
- Manage lead status dynamically.
- Persistent storage using JSON file (`leads.json`) on the server.
- Advanced JavaScript with ES6 Classes for clean architecture.
- Responsive design for all screen sizes.

## Tech Stack
- Node.js (Express.js server)
- HTML5, CSS3, JavaScript
- JSON for data storage
- Render.com deployment for live back-end functionality
