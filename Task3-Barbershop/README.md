# Task 3 – Solo Barbershop Website

## Overview
This project is a **professional barbershop website** built during the FUTURE_FS_01 Internship.  
It highlights services, gallery, and online booking functionality for a barbershop.

## Features
- Hero section with engaging background and call-to-action.
- Services section with pricing for haircuts and grooming.
- Gallery section with responsive grid images.
- Contact form integrated with Formspree to receive bookings via email.
- Responsive navigation bar and mobile menu support.
- Modern design with dark theme and accent colors for visual appeal.

## Tech Stack
- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript (for interactive elements and animations)
- Formspree for booking form submissions
- Unsplash & free online images for gallery
