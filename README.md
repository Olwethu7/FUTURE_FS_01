FUTURE_FS_01 Internship Projects

This repository contains all the tasks completed for the internship program. It demonstrates web development skills, including HTML, CSS, JavaScript, Node.js, and deployment on platforms like Netlify and Render.

Live Project Links
Task 1 - Personal Portfolio
Description:
A professional portfolio website showcasing personal information, skills, projects, and contact information. Designed to be fully responsive and visually appealing, with smooth navigation and interactive sections.
Technologies: HTML, CSS, JavaScript, responsive design.
Live Link: [View Task 1 Portfolio](https://olwethu-cwele-task1-portfolio.netlify.app/)

Task 2 - Client Lead Management System (CRM)
Description:
A fully functional CRM system to manage client leads. Users can add, view, and store leads. Backend implemented using Node.js, while the frontend is HTML, CSS, and JavaScript. The system stores data in a JSON file and is accessible online.
Technologies: Node.js, Express.js, HTML, CSS, JavaScript, REST API.
Live Link: [View Task 2 CRM](https://future-fs-01-wv25.onrender.com)

Task 3 - Professional Barbershop Website
Description:
A modern, advanced barbershop website for New Legacy Barbershop. The site focuses exclusively on haircuts, including services, gallery, and a booking form that sends inquiries via email. The site uses a dark professional theme, responsive design, and smooth animations.
Technologies: HTML, CSS, JavaScript, responsive grid layouts, email form via Formspree.
Live Link: [View Task 3 Barbershop Website](https://olwethu-cwele-task3-barbershop.netlify.app/)
