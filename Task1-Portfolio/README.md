# Task 1 – Portfolio Website

## Overview
This project is my **professional portfolio website** built during the FUTURE_FS_01 Internship.  
It showcases my skills, projects, and personal profile in a clean, responsive, and modern design.

## Features
- Fully responsive design for desktop, tablet, and mobile.
- Modern UI with CSS Grid and Flexbox layouts.
- Smooth scrolling navigation and animated sections.
- Portfolio section to showcase projects with links and descriptions.
- Contact form integrated with Formspree for receiving messages.
- Professional color scheme and typography for visual appeal.

## Tech Stack
- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript (for interactive elements)
- Google Fonts (Roboto)
